1  R 语言入门

R语言最初由Ross Ihaka和Robert Gentleman于1993年开发，是从S语言演变而来的开源编程语言。因其强大的数据处理和统计分析能力，R已成为数据科学领域的标准工具之一。

1.1 R在数据科学中的优势

  • 丰富的统计与数据分析功能：内置众多统计分析函数，涵盖回归分析、时间序列分析、聚类分析等
  • 强大的数据可视化能力：通过ggplot2等包可生成出版级质量的图表
  • 广泛的社区支持与软件包生态：CRAN和Bioconductor提供数以万计的扩展包

library(dplyr)
library(ggplot2)

# 生成示例数据
set.seed(123)
demo_data <- data.frame(
  x = rnorm(100),
  y = rnorm(100)
)

# 描述性统计

  summarise(
    mean_x = mean(x),
    mean_y = mean(y),
    sd_x = sd(x),
    sd_y = sd(y)
  )
print(summary_stats)
#>       mean_x     mean_y      sd_x      sd_y

# 线性回归模型

1.4.2 ggplot2包

Tip

ggplot2基于”图形语法”理论，通过层次化的方式创建图形。

1.4.2.1 基本用法

library(ggplot2)

# 创建示例数据
scatter_data <- data.frame(
  x = rnorm(100),
  y = rnorm(100),
  group = sample(letters[1:3], 100, replace = TRUE)
)

# 散点图
ggplot(scatter_data, aes(x = x, y = y, color = group)) +
  geom_point(size = 3, alpha = 0.7) +
  labs(title = "散点图示例", x = "X轴", y = "Y轴") +
  theme_bw()

1.4.2.2 常见图表类型

# 折线图
time_data <- data.frame(
  time = 1:50,
  value = cumsum(rnorm(50))
)

p1 <- ggplot(time_data, aes(x = time, y = value)) +
  geom_line(color = "blue") +
  labs(title = "折线图") +
  theme_bw()

# 直方图
p2 <- ggplot(scatter_data, aes(x = x)) +
  geom_histogram(binwidth = 0.5, fill = "green", alpha = 0.6) +
  labs(title = "直方图") +
  theme_bw()

# 箱线图
p3 <- ggplot(scatter_data, aes(x = group, y = x, fill = group)) +
  geom_boxplot() +
  labs(title = "箱线图") +
  theme_bw()

cowplot::plot_grid(p1, p2, p3, labels = "auto", ncol = 3)

1.4.2.3 图层叠加

1.4.3 stringr包

library(stringr)

text <- "Hello, R language!"

# 提取子字符串
str_sub(text, 8, 9)
#> [1] "R "
# 拼接字符串
str_c(text, " It's powerful!")
#> [1] "Hello, R language! It's powerful!"
# 替换字符串
str_replace(text, "language", "world")
#> [1] "Hello, R world!"
# 正则表达式匹配
text_vec <- c("apple123", "banana456", "cherry789")
str_detect(text_vec, "\\d+")
#> [1] TRUE TRUE TRUE
str_replace_all(text_vec, "\\d+", "")
#> [1] "apple"  "banana" "cherry"

1.4.4 tidyr包

library(tidyr)

# 创建宽格式数据
wide_data <- data.frame(
  ID = c(1, 2, 3),
  Math = c(90, 85, 88),
  Science = c(80, 78, 95),
  English = c(85, 89, 92)
)

# 宽转长
long_data <- wide_data |>
  pivot_longer(
    cols = Math:English,
    names_to = "Subject",
    values_to = "Score"
  )
print(long_data)
#> # A tibble: 9 × 3
#>      ID Subject Score
#>   <dbl> <chr>   <dbl>
#> 1     1 Math       90
#> 2     1 Science    80
#> 3     1 English    85
#> 4     2 Math       85
#> 5     2 Science    78
#> 6     2 English    89
#> 7     3 Math       88
#> 8     3 Science    95
#> 9     3 English    92
# 长转宽
wide_data_again <- long_data |>
  pivot_wider(
    names_from = Subject,
    values_from = Score
  )
print(wide_data_again)
#> # A tibble: 3 × 4
#>      ID  Math Science English
#>   <dbl> <dbl>   <dbl>   <dbl>
#> 1     1    90      80      85
#> 2     2    85      78      89
#> 3     3    88      95      92
Note

pivot_longer()和pivot_wider()是tidyr包的核心函数，分别用于宽长格式转换。这两个函数在数据清洗的时候作用很大，是必须要掌握的两个工具。

1.4.5 数据检查和清洗

# 创建含缺失值的数据框
data_with_missing <- data.frame(
  ID = c(1, 2, 3, 4),
  Name = c("Alice", "Bob", "Charlie", "David"),
  Score = c(95, NA, 88, 92)
)

# 检查缺失值
cat("各列缺失值数量：\n")
#> 各列缺失值数量：
colSums(is.na(data_with_missing))
#>    ID  Name Score 
#>     0     0     1
# 查看完整行
cat("\n完整行（无缺失值）：\n")
#> 
#> 完整行（无缺失值）：
data_with_missing[complete.cases(data_with_missing), ]
#>   ID    Name Score
#> 1  1   Alice    95
#> 3  3 Charlie    88
#> 4  4   David    92
# 删除含缺失值的行
cat("\n删除缺失值后：\n")
#> 
#> 删除缺失值后：
tidyr::drop_na(data_with_missing)
#>   ID    Name Score
#> 1  1   Alice    95
#> 2  3 Charlie    88
#> 3  4   David    92
# 填充缺失值（用均值填充）
cat("\n用均值填充缺失值后：\n")
#> 
#> 用均值填充缺失值后：
data_with_missing |>
  mutate(Score = ifelse(is.na(Score), mean(Score, na.rm = TRUE), Score))
#>   ID    Name    Score
#> 1  1   Alice 95.00000
#> 2  2     Bob 91.66667
#> 3  3 Charlie 88.00000
#> 4  4   David 92.00000
Note

处理缺失值的常用策略包括：删除含缺失值的行（drop_na()）、用均值/中位数填充（replace_na()）、或使用插值方法。选择哪种方法取决于缺失数据的比例和缺失机制。
